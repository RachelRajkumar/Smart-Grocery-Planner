🛒 Smart Grocery Planner

✨ A simple and smart frontend web application to manage grocery items, track budget, and calculate total expenses.

🚀 Features

✅ Add grocery items (Name + Price)  
 🗑 Delete items easily  
🔍 Search products instantly  
💰 Set your budget  
🧾 Auto calculate total cost  
💵 Show remaining balance  
💾 Data saved using LocalStorage (persists after refresh)  
📱 Fully responsive design  

🛠️ Tech Stack

🌐 HTML5 – Structure  
🎨 CSS3 – Styling  
⚙️ JavaScript (Vanilla JS) – Logic  
💾 LocalStorage – Data storage
